# Kingdom & Company Landing Page

Modern responsive landing page built with React, TypeScript, and Framer Motion.

## Live Demo
https://kingdom-company-landing.vercel.app/

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Mapbox GL

## Features
- Smooth scroll animations
- Responsive design
- Animated typography
- Interactive FAQ & agenda
- Popup form with validation
- Mapbox location map
